# Unit Converter(Meter,litre,kilogram)

A simple Metric/Imperial Unit Converter built with HTML, CSS, and JavaScript.

## About

This project lets a user enter a number and convert it between common metric and imperial units. When the Convert button is clicked, the page displays conversions for length, volume, and mass.

## Features

- Converts meters to feet and feet to meters
- Converts liters to gallons and gallons to liters
- Converts kilograms to pounds and pounds to kilograms
- Takes user input from the page
- Updates conversion results with JavaScript
- Styled with a simple responsive card layout

## Technologies Used

- HTML
- CSS
- JavaScript

## Project Files

- `index.html` - contains the page structure
- `index.css` - contains the styling
- `index.js` - contains the conversion logic
- `README.md` - project information

## How To Run

Open `index.html` in your browser.

You do not need to install any dependencies for this project.

## How To Use

1. Enter a number in the input box.
2. Click the Convert button.
3. View the converted values for length, volume, and mass.

## Conversion Rates

```text
1 meter = 3.281 feet
1 liter = 0.264 gallon
1 kilogram = 2.204 pounds
```
